# Android qr code scanner

### Description
 Full stable QR code scanner in android.

### Functionality 
 User can:
 ⋅⋅* use flashlight 
 ⋅⋅* search in the internet with scanned qr code value
 ⋅⋅* copy to clipboard qr code value
 ⋅⋅* share qr code value
 ⋅⋅* see own history of scanned qr codes
